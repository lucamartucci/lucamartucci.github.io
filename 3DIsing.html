Introduction¶

We simulate the 3D Ising model on an N x N x N cubic lattice using the Metropolis Monte Carlo algorithm. We compute the magnetization per spin as a function of temperature for different lattice sizes as the main observable to monitor, which will make manifest the phase transition which is the characteristic feature of this model.

We define some functions to compute quantities of interest, namely:

$\cdot$ the energy of the configuration, given by the Hamiltonian: $$H=-J\sum_{\langle i, j\rangle}S_iS_j$$ where we sum on neighboring 'neurons';

$\cdot$ the difference in energy $\Delta E$ when flipping a single spin: i.e. $S_i \longrightarrow -S_i$. Notice that, after a single flip spin, we return the difference in energy $$\Delta E=2JS_i\sum_{\langle i, j\rangle}S_j$$ in fact, before: $E_{\text{init.}}=-JS_i\sum_{\langle i, j\rangle}S_j$ and after: $E_{\text{fin.}}=-J(-S_i)\sum_{\langle i, j\rangle}S_j$, hence the expression;

$\cdot$ the average magnetization of each configuration, namely, the mean of the total spin sum;

$\cdot$ the autocorrelation function, i.e., the weighted covariance between the atoms.

# JIT-compiled energy difference and update

@jit(nopython=True)
def compute_energy(lattice, J):
    E = 0.0
    N = lattice.shape[0]
    for x in range(N):
        for y in range(N):
            for z in range(N):
                S = lattice[x,y,z]
                E -= J * S * (
                    lattice[(x+1)%N,y,z] + lattice[x,(y+1)%N,z] + lattice[x,y,(z+1)%N]
                )
    return E

@jit(nopython=True)
def delta_energy(lattice, x, y, z, J):
    N = lattice.shape[0]
    S = lattice[x, y, z]
    neighbors = (
        lattice[(x+1)%N, y, z] + lattice[(x-1)%N, y, z] +
        lattice[x, (y+1)%N, z] + lattice[x, (y-1)%N, z] +
        lattice[x, y, (z+1)%N] + lattice[x, y, (z-1)%N]
    )
    return 2 * J * S * neighbors

@jit(nopython=True)
def compute_magnetization(lattice):
    return np.abs(lattice.sum()) / lattice.size

def autocorrelation(data):
    n = len(data)
    data = np.array(data) - np.mean(data)
    var = np.var(data)
    acf = np.correlate(data, data, mode='full')[n-1:] / (var * np.arange(n, 0, -1))
    return acf

The following function does a single sweep of the Metropolis algorithm. At each iteration it picks a spin uniformly at random and looks at the change in energy. The spin is then flipped according to the Metropolis rule:

$\cdot$ if the energy is decreased, i.e. $\Delta E<0$, we always accept the proposed move;

$\cdot$, we accept the move with probability $\exp(-\Delta E/T)$.

@njit
def metropolis_sweep(lattice, beta, J, xs, ys, zs, us):
    N = lattice.shape[0]
    for i in range(N**3):
        x = xs[i]
        y = ys[i]
        z = zs[i]
        # compute ΔE via jit‐ted delta_energy
        dE = delta_energy(lattice, x, y, z, J)
        if dE <= 0 or us[i] < math.exp(-beta * dE):
            lattice[x, y, z] = -lattice[x, y, z]
    return lattice

Simulation Function¶

Here we actually simulate the Metropolis method. First, we perform some equilibration steps to 'average out' the noise induced by the arbitrary initial state in which the system is found. After these equilibration steps, that let the system "forget" the initial condition, the configuration is statistically typical of the Boltzmann distribution for that temperature. Then, for each sweep of the algorithm, we store the magnetization and energy, which we will use to compute the following observables:

$\cdot$ Average magnetization: which we will expect, in our visual analysis, to incur in a phase transition around the critical tempoerature for the 3D Ising model.

$\cdot$ Average energy: this would have to increase proportionally to the temperature, stabilizing after the phase transition.

$\cdot$ Susceptibility $\chi=(\langle M^2\rangle-\langle M\rangle^2)/T$: this weighted magnetization variance measures how much the magnetization of the system responds to a small external magnetic field. Even when no external field is applied, it gives insights into fluctuations in magnetization and critical behavior near the phase transition.

def simulate_ising_3d(N, T, n_eq=500, n_steps=2500, J=1.0):
    rng = np.random.default_rng(SEED)
    lattice = rng.choice(np.array([-1, 1]), size=(N, N, N))
    beta = 1.0 / T

    # pre-generate all random choices for both equilibration and measurement
    total_sweeps = n_eq + n_steps
    n_trials = total_sweeps * N**3
    xs = rng.integers(0, N, size=n_trials)
    ys = rng.integers(0, N, size=n_trials)
    zs = rng.integers(0, N, size=n_trials)
    us = rng.random(n_trials)

    # Equilibration
    for sweep in range(n_eq):
        i0 = sweep * N**3
        lattice = metropolis_sweep(
            lattice, beta, J,
            xs[i0:i0+N**3], ys[i0:i0+N**3],
            zs[i0:i0+N**3], us[i0:i0+N**3]
        )

    # Observables
    mags = np.empty(n_steps)
    energies = np.empty(n_steps)
    states = []

    for sweep in range(n_steps):
        i0 = (n_eq + sweep) * N**3
        lattice = metropolis_sweep(
            lattice, beta, J,
            xs[i0:i0+N**3], ys[i0:i0+N**3],
            zs[i0:i0+N**3], us[i0:i0+N**3]
        )
        mags[sweep] = compute_magnetization(lattice)
        energies[sweep] = compute_energy(lattice, J) / lattice.size
        if sweep in [n_steps//4, n_steps//2, 3*n_steps//4]:
            states.append(lattice.copy())

    # Thermodynamic averages
    avg_M = mags.mean()
    avg_E = energies.mean()

    # Susceptibility
    chi = (np.var(mags) * N**3) / T 

    # Autocorrelation
    acf_M = autocorrelation(mags)
    
    # Distribution histograms
    hist_E, bins_E = np.histogram(energies, bins=50, density=True)
    hist_M, bins_M = np.histogram(mags, bins=50, density=True)

    return  avg_M, avg_E, chi, acf_M, (hist_E, bins_E), (hist_M, bins_M), states

We now actually run the simulation for choices $N=10, 20, 30$ and a range of temperature centered at the known critical tempearature $T_C\approx 4.5$, and store the results.

Ns = [10, 20, 30]
temperatures = np.linspace(3.5, 5.0, 15)

# Run simulations in parallel over (N, T) combinations
def run_for_params(N, T):
    avg_M, avg_E, chi, acf_M, (hist_E, bins_E), (hist_M, bins_M), states = simulate_ising_3d(N, T)
    return (N, T, avg_M, avg_E, chi, acf_M, (hist_E, bins_E), (hist_M, bins_M), states)

# Flatten parameter list
tasks = [(N, T) for N in Ns for T in temperatures]

# Execute in parallel using all available cores
results_list = Parallel(n_jobs=4)(delayed(run_for_params)(N, T) for N, T in tqdm(tasks))

At the critical temperature, the system undergoes a spontaneous symmetry-breaking transition, switching from a disordered phase (random spins) to an ordered one (aligned spins). The system is extremely sensitive to perturbations (a small field or fluctuation can tip the balance) causing huge fluctuations in magnetization, coexistence of many domains of different magnetization and spins correlated over large distances (divergent correlation length). In the last plot, this intuitive description is backed by the sharp behavior of the susceptibility around the critical temperature: when the phase transition happens, the variation is very large, otherwise, for all other temperatures, it stays very homogeneous.

Here we see the evolution of the autocorrelation of magnetization. When going thorugh a phase transition, from example going from solid to liquid state, a solid phase is one in which the correlations of positions are very long range, in fact they persist on arbitrary large distance: if you know the crystal structure of a solid locally, you can guess the average position of an atom even if it is very far away. Similarly here, autocorrelation decays exponentially as expected and we observe the critical slow down sensibly near to $T_c$.

Conclusions¶

Through Monte Carlo simulations of the 3D Ising model, we successfully observed the emergence of a phase transition near the critical temperature $T_c$. The magnetization, energy, and their fluctuations displayed clear signatures of this transition, including critical slowing down and broader energy distributions. Our analysis confirms that the model captures the expected physical behavior.
